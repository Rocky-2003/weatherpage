# Weather Webpage

Welcome to the Weather Webpage, a simple and intuitive web application that provides real-time weather conditions for a specified city. Stay informed about the weather with just a few clicks!

## Features

- **City-based Weather Information:** Enter the name of the city you want to know about, and the webpage will display the current weather conditions.

- **Temperature, Humidity, and Wind Speed:** Get detailed information about the temperature, humidity, and wind speed in the selected city.

- **Visual Representation:** The weather information is complemented by visual icons and graphics to enhance user understanding.

## How to Use

1. **Access the Webpage:**
   Open the `index.html` file in your web browser to access the Weather Webpage.

2. **Enter City Name:**
   Type the name of the city you want to check the weather for into the designated input field.

3. **Retrieve Weather Information:**
   Click the "Get Weather" button to retrieve real-time weather information for the specified city.

4. **View Results:**
   The webpage will display the current temperature, humidity, wind speed, and additional weather details for the entered city.

## Technologies Used

- **HTML:** Provides the structure of the web page.
- **CSS:** Enhances the visual styling and layout.
- **JavaScript:** Interacts with a weather API to fetch and display real-time weather information.

## Project Structure

- `body.html`: The main HTML file containing the structure of the web page.
- `style.css`: The CSS file for styling and layout.
- `script.js`: The JavaScript file containing the logic to interact with the weather API.

This project utilizes a weather API to fetch real-time weather data. You may need to obtain an API key from a weather service provider and replace it in the `script.js` file.
